# 音乐 APP 原型设计提示词

## 一、项目概述
使用 Pencil 工具设计一款音乐 APP 的原型，应用名称自拟（如“乐听”）。风格偏向现代、简洁、深色或浅色均可。需包含以下页面和功能。

## 二、页面列表与功能点

### 1. 启动页 / 引导页（可选）
- 展示 APP Logo 和加载进度
- 2-3 张引导卡片（仅首次启动显示），介绍核心功能（海量曲库、个性推荐、离线听歌）
- 点击“立即体验”进入首页

### 2. 登录 / 注册页
- **手机号 / 邮箱输入框**
- **密码输入框**（可切换显示/隐藏）
- **登录按钮** → 成功后跳转至首页
- **注册按钮** → 跳转至注册页（包含昵称、手机号、验证码、密码）
- **第三方登录图标**（微信、QQ、Apple ID）
- **“跳过”按钮** → 以游客模式进入首页（部分功能受限）

### 3. 首页（推荐 Tab）
- **顶部导航栏**：左侧汉堡菜单（打开侧边栏），右侧搜索图标（跳转搜索页）
- **轮播图**（3-5 张，点击跳转至对应歌单/活动页）
- **功能入口区**：一行图标按钮（每日推荐、排行榜、歌单、电台）
- **推荐歌单**：横向滚动列表，每项显示封面、标题、播放量。点击进入歌单详情页
- **最新音乐**：纵向列表，每行显示歌曲封面、歌名、歌手、更多操作（···）。点击歌曲进入播放器页
- **底部 Tab Bar**：首页、搜索、播放器（迷你栏）、我的

### 4. 搜索页
- **搜索输入框**：自动聚焦，支持实时联想（展示热门搜索词）
- **历史记录**：展示最近 5 条搜索词，支持清空
- **热门搜索**：推荐的热门关键词（可点击）
- **搜索结果页**（输入后替换当前视图）：
  - 按分类展示：单曲、歌单、歌手、专辑
  - 每个结果项可点击：单曲→播放器页，歌单→歌单详情页，歌手→歌手详情页
- **返回按钮**：回到首页

### 5. 播放器页（核心页面）
- **顶部**：歌曲封面大图（带唱片机样式或动态效果）、歌名、歌手、更多（分享/评论）
- **中部**：进度条（可拖拽），已播放时长/总时长
- **底部操作区**：
  - 喜欢（红心）按钮
  - 下载按钮
  - 评论按钮（跳转评论页）
  - 列表循环/随机播放切换图标
- **播放控制区**：上一首、播放/暂停、下一首（大按钮）
- **歌词显示**（可选）：可收起/展开，当前歌词高亮
- **底部迷你条**（退出播放器页后出现在 Tab Bar 上方）：显示当前播放歌曲封面、歌名、播放/暂停按钮，点击回到播放器页

### 6. 歌单详情页
- **顶部背景图**：歌单封面大图 + 歌单标题、创建者、简介、收藏数、分享数
- **操作按钮**：全部播放（跳转播放器页并播放第一首）、收藏歌单（红心变实心）
- **歌曲列表**：
  - 每行：序号/红心图标、歌名、歌手、时长
  - 点击歌曲→跳转播放器页并播放该曲
  - 右侧更多菜单（添加到下一首、下载、评论）
- **返回按钮**：回到上一页

### 7. 歌手详情页
- **顶部**：歌手照片、歌手名、关注按钮、粉丝数
- **Tab 切换**：热门歌曲、专辑、相似歌手
- **热门歌曲列表**：点击直接播放（跳转播放器）
- **专辑列表**：横向滚动，点击进入专辑详情页（结构与歌单详情类似）
- **返回按钮**

### 8. 我的页面（个人中心）
- **用户头像、昵称、等级/会员标识**（点击进入个人资料编辑页）
- **数据统计**：动态、粉丝、关注数（可点击查看列表）
- **功能菜单列表**：
  - 我的歌单（包括创建的歌单、收藏的歌单）→ 跳转至我的歌单管理页
  - 最近播放（跳转最近播放列表页）
  - 本地音乐（扫描本地存储）
  - 设置（跳转设置页：主题、音质、缓存、清除缓存、退出登录）
- **退出登录按钮**

### 9. 侧边栏（从首页左侧滑出）
- 用户头像、昵称
- 夜间模式切换开关
- 消息中心（红点提示）
- 音乐闹钟、听歌识曲等工具入口
- 客服与反馈

## 三、页面路由关系

- **启动页** → 登录页（或引导页后）
- **登录页** → 首页（登录成功/跳过）
- **首页** → 搜索页（点击搜索图标）
- **首页** → 歌单详情页（点击推荐歌单项）
- **首页** → 播放器页（点击最新音乐列表中的歌曲）
- **搜索页** → 搜索结果页（输入关键词自动跳转）
- **搜索结果页** → 歌单详情页 / 歌手详情页 / 播放器页
- **播放器页** → 歌单详情页（点击“下一首”时可能涉及，但通常从播放器页可返回上一页）
- **歌单详情页** → 播放器页（点击“全部播放”或点击任意歌曲）
- **歌手详情页** → 播放器页（点击热门歌曲）
- **我的页面** → 歌单管理页 / 最近播放页 / 设置页
- **设置页** → 可返回我的页面
- **底部 Tab Bar**：首页、搜索、播放器（注：第三个 Tab 通常直接打开播放器页，但为了简化，可设计成点击 Tab 展开迷你播放器或全屏播放器页；更常见做法：第三个 Tab 是“播放器”图标，点击进入播放器页，同时底部有迷你条）
- **迷你播放条**：在任何非播放器页的底部可见（Tab Bar 上方），点击回到播放器页

## 四、交互与状态说明

- 所有列表支持下拉刷新、上拉加载更多
- 播放器页的“喜欢”红心状态需同步到歌单详情页的歌曲列表中（红心显示）
- 搜索页需要提供取消/清空输入功能
- 播放器页退出后，底部迷你条保持显示当前播放状态
- 侧边栏可通过左滑或点击首页顶部汉堡菜单打开
- 页面转场使用 Pencil 预设的“向右滑动”或淡入淡出效果

## 五、使用 Pencil 绘制的具体要求

- 使用 Pencil 的“移动模板”（Phone 尺寸，如 375×667）
- 为每个页面单独创建一个画板（Page）
- 在画板之间用“连接线”标明跳转关系（如从登录页连接到首页）
- 标注关键组件的交互说明（如“点击此处跳转”）
- 建议使用共享样式（颜色、字体、按钮）保持设计一致性
- 输出格式：.ep 文件（Pencil 工程文件）或导出为 PDF/PNG

## 六、优先级（MVP 版本可先实现）

P0（必须有）：登录/注册页、首页（推荐）、播放器页、我的页面、底部 Tab Bar、迷你播放条  
P1（重要）：搜索页、歌单详情页、歌手详情页、侧边栏  
P2（可后续）：启动引导页、设置页、本地音乐、消息中心
