# TationCloud 原型计划

日期：2026-04-06

## 当前原型状态
- 已形成桌面端与移动端两套主流程原型，默认以浅色模式展示。
- 已补齐主题变量体系，支持浅色 / 深色模式扩展；当前交付重点为默认浅色体验与统一壳层。
- 本轮重点完善了桌面端统一顶栏、消息铃铛与用户信息、文件浏览结构、移动端底部导航，以及移动端通知 / 账单 / 版本历史页面。

## 已完成原型
- 桌面端：首页、文件、搜索、文件预览 / 解锁、上传中心、会员中心、设置、分享、回收站、通知中心、传输任务、版本历史、账单与订单、共享给我、文件夹管理、登录 / 注册 / 找回密码、状态页。
- 移动端：首页、文件、预览 / 解锁、上传中心、分享管理、会员中心、设置、回收站、通知中心、账单与订单、版本历史、登录页。
- 桌面统一能力：品牌侧栏、空间使用进度、统一顶栏搜索、铃铛消息数、头像 + 套餐信息、VIP / SVIP 语义统一。
- 移动统一能力：4 栏主入口底部导航、居中上传主动作、统一品牌头部、同一套卡片与信息层级。
- 产品能力覆盖：中文 / English 切换、主题切换入口、文件加密与待解锁状态、分享权限、账单记录、版本恢复语义、回收站恢复、批量操作提示。

## 核心流程
- 桌面端：首页总览 → 文件列表 → 文件夹管理 → 文件预览 / 解锁 → 上传加密 → 会员 / 账单 → 分享 → 通知 → 版本历史。
- 移动端：首页总览 → 文件网格 → 预览 / 上传 → 分享 → 会员 / 账单 → 设置 → 通知 / 版本历史。
- 文件浏览已调整为更符合用户习惯的结构：
  - 桌面端采用目录树 + 排序筛选 + 文件卡片网格 + 详情区。
  - 移动端采用目录上下文 + 排序筛选 + 双列卡片网格 + 长按批量提示。

## 组件与主题
- 已升级共享组件：Sidebar、Topbar、Mobile Nav。
- 已新增共享组件：Desktop File Card、Mobile File Card、Notice Badge。
- 顶栏消息入口统一为铃铛样式，并使用小红点 + 数字表示未读数量。
- 主题变量已覆盖主要表面、字段、弱化层、边框与前景色，默认浅色模式下所有核心流程页面均可直接查看。
